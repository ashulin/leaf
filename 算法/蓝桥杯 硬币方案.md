## 硬币方案

>   有50枚硬币，可能包括4种类型：1元，5角，1角，5分。
>   已知总价值为20元。求各种硬币的数量。
>   比如：2,34,6,8 就是一种答案。

暴力解法：

```java
    @Test
    public void coinCombination() {
        int count = 0;
        int loop = 0;
        for (int i = 0; i <= 20; i++) {
            for (int j = 0; j <= 40; j++) {
                for (int k = 0; k <= 50; k++) {
                    loop++;
                    int l = 50 - i - j - k;
                    if (l < 0) {
                        break;
                    }
                    if (i * 100 + j * 50 + k * 10 + l * 5 == 2000) {
                        count++;
                    }
                }
            }
        }
        // count:50  loop:19106
        log.info("count: {}, loop: {}", count ,loop);
    }
```

优化方案：

```java
    @Test
    public void coinCombination() {
        int count = 0;
        int loop = 0;
        // 5*32 + 100*18 < 2000，5*30 + 100*20 > 2000，5分的个数小于32
        // 5分硬币的个数一定为偶数: 2i
        for (int i = 0; i < 16; i++) {
            // 10*34 + 100*16 < 1960，10*33 + 100*17 > 2000，1角的个数小于34
            // 1角硬币与5分硬币组合一定为5角的倍数
            for (int j = (5 - (i % 5)) % 5; j < 34; j = j + 5) {
                loop++;
                int k = 50 - 2 * i - j;
                // 硬币数量过多
                if (k < 0) {
                    break;
                }
                int result = k * 10 + j + i;
                if (result >= 200
                        // 5*30 + 10*33 = 480 < 2000, 即一定需要1元与5角
                        // 使用5角代替1元，削减总额至20元
                        // 确定5角个数未超过替换前1元的个数，1元个数不超过20
                        && (result - 200) / 5 <= k) {
                    count++;
                }
            }
        }
        // count:50  loop:105
        log.info("count: {}, loop: {}", count ,loop);
    }
```
